# Loan Approval Prediction

## Project Overview
This project aims to predict whether a bank loan application will be approved based on customer characteristics such as credit score and salary.

The objective is to explore the data, understand relationships between variables, and build predictive models capable of estimating the likelihood of loan approval.

## Dataset
The dataset contains information about loan applicants, including:
- Credit score
- Salary
- Loan approval status

These variables help determine whether a loan request is likely to be accepted or rejected.

## Project Steps

### 1. Exploratory Data Analysis (EDA)
The first step consisted of performing a descriptive analysis to better understand the dataset.

This included:
- examining the distribution of variables
- identifying patterns in the data
- visualizing relationships using charts and graphs

The goal was to gain insights into how the data behaves before building predictive models.

### 2. Predictive Modeling
Two types of models were developed:

- **Regression model**
- **Classification model**

These models were trained to predict whether a loan request would be approved.

### 3. Model Evaluation
The models were evaluated using **accuracy** as the main performance metric.

Accuracy measures the proportion of correct predictions made by the model.

The model with the **highest accuracy** was selected as the final prediction model.
